# ChiCTR临床试验采集 - 快速开始指南

## ⚠️ 重要说明

由于ChiCTR和CDE网站有强大的WAF防护，以下是推荐的采集策略：

## 方法1：使用WebFetch手动采集（推荐）

### ChiCTR - 按基因搜索

```
WebFetch URL: https://www.chictr.org.cn/searchproj.html
(POST表单数据)
- regname: [基因名称]  # 例如: KRAS, EGFR, PD-1
- studytpe: 干预性研究
- recruit: 正在招募,尚未开始
- page: 1
```

**示例 - 搜索KRAS基因的干预性研究**:

1. 打开浏览器访问: https://www.chictr.org.cn/searchproj.html
2. 点击"更多筛选"
3. 在"注册题目"输入: KRAS
4. "研究类型"选择: 干预性研究
5. "征募状态"选择: 正在招募, 尚未开始
6. 点击搜索
7. 获取第1页结果（10条）
8. 翻页获取第2、3、...页

### CDE - 按适应症搜索

```
URL: https://www.chinadrugtrials.org.cn/clinicaltrials.prosearch.dhtml?pro=y
参数:
- keyword: [肿瘤类型或基因]  # 例如: 肺癌, EGFR, PD-1
- pro: y
- pageNum: 1
```

## 采集流程建议

### 建议的采集顺序（逐个基因）:

1. **第1批（核心靶点）**: EGFR, PD-1, PD-L1, KRAS, HER2
2. **第2批（重要靶点）**: ALK, BRAF, MET, RET, FGFR
3. **第3批（其他靶点）**: NTRK, BRCA, PARP, CDK4/6, PIK3CA

### 每个基因的采集步骤:

1. 搜索基因名称（如：EGFR）
2. 获取第1页结果（10条）
3. 逐个访问详情页
4. 翻页获取第2、3、...页
5. 等待2-3秒
6. 开始下一个基因

## 故障排除

### 问题1: POST请求被拦截

**解决方案**: 
- 使用WebFetch工具手动采集
- 或使用示例数据

### 问题2: 获取不到结果

**解决方案**:
- 检查搜索条件是否正确
- 尝试不同的基因名称
- 使用更通用的关键词

### 问题3: WAF封禁

**解决方案**:
- 等待一段时间后再尝试
- 使用不同的IP地址
- 减少请求频率
